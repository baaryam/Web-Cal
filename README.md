# Web-Cal

🧮 Functional Web-Based Calculator

A simple, responsive, and functional web-based calculator built using HTML, CSS, and JavaScript for the course INTE 21323 – Web Applications Development.

Designed and developed by Mucksith Baary (IM/2022/133) – Year 2, Semester 1.

📌 Project Overview

This project is a basic calculator web application that performs standard arithmetic operations:

Addition (+)
Subtraction (−)
Multiplication (×)
Division (÷)

The calculator features a clean UI design, hover effects, button animations, and mobile responsiveness.

🚀 Features

✔️ Perform basic arithmetic operations
✔️ Clean and modern user interface
✔️ Responsive design (Desktop + Mobile)
✔️ Hover and active button effects
✔️ Error handling for invalid expressions
✔️ Clear (C) button functionality

🛠️ Technologies Used

HTML5 – Structure of the calculator

CSS3 – Styling, layout, and responsiveness

JavaScript (Vanilla JS) – Calculator logic and functionality

💻 How It Works

Numbers and operators are appended to the display using the appendToDisplay() function.

The calculateResult() function evaluates the mathematical expression using JavaScript's eval() function.

The clearDisplay() function resets the display.

Basic error handling prevents the application from crashing on invalid inputs.

📱 Responsive Design

The calculator is fully responsive and adjusts its layout for:

Large screens (Desktop)
Medium screens (Tablets)
Small screens (Mobile devices under 480px)

Media queries are used to optimize button sizes, spacing, and layout.
